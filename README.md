# BigNumber.php

A PHP package for arbitrary-precision decimal and non-decimal arithmetic inspired by bignumber.js

Installation
----

``` bash
composer require kabbouchi/bignumber
```
## Usage

```php
use KABBOUCHI\BigNumber;

BigNumber.of(123).plus(12).div(1234).toString();
```
